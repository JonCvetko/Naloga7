# README

## Glavne funkcionalnosti spletne aplikacije

1. **Avtentikacija uporabnikov**  
   Omogoča registracijo, prijavo in upravljanje uporabniških sej z uporabo JWT.

2. **Nadzor omrežnih naprav**  
   Aplikacija skenira lokalno omrežje, zaznava povezane naprave ter prikazuje njihove lastnosti in status.

3. **Analiza varnostnih ranljivosti**  
   Prepozna varnostna tveganja, napake in ranljivosti na omrežnih napravah ter uporabniku predlaga ustrezne rešitve.

4. **Administratorski nadzor**  
   Omogoča upravljanje uporabnikov, njihovih vlog (trgovec, kupec, admin) ter dostopov.

5. **Vizualni pregled omrežja**  
   Vključuje grafični prikaz povezav v omrežju in omogoča interaktivni pregled trenutnega stanja sistema.
